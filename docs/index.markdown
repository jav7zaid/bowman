# Bowman

### A simple static site generator powered by Node.js, Webpack, and Browsersync

## Getting Started

Install:

        npm i -g @waldronmatt/bowman

Create project:

       create-bowman-starter -y
        
Install dependencies:

        npm install

Run dev environment:

        npm run dev

Build and serve for production:

        npm run build
